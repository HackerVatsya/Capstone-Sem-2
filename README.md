🎮 GAME LIBRARY EXPLORER 🎮
=========================

Game Library Explorer is a high-performance, responsive web application designed for enthusiasts to discover, search, and manage a personalized list of video games. This project serves as a comprehensive demonstration of modern front-end development capabilities using pure Vanilla JavaScript, HTML5, and CSS3, without the reliance on heavyweight frameworks.

The application interfaces with the RAWG Video Games Database API, providing real-time access to a vast repository of over 500,000 titles.

--------------------------------------------------------------------------------

🎯 KEY OBJECTIVES
-----------------
- DATA INTEGRATION: Successfully fetch and process asynchronous data from a third-party REST API.
- DYNAMIC UI: Implement a reactive user interface that updates instantly based on user input.
- STATE MANAGEMENT: Utilize browser localStorage to persist user preferences (Favorites) across sessions.
- PERFORMANCE: Optimize DOM manipulation using efficient array methods (map, filter, sort) to handle dozens of game records smoothly.

--------------------------------------------------------------------------------

🚀 CORE FEATURES
----------------

1. SEARCH
   Real-time filtering of game titles with instant feedback.

2. GENRE FILTER
   Dynamic filtering using categories like Action, RPG, and Strategy.

3. SMART SORT
   Multiple sorting algorithms implemented for Rating and Alphabetical order.

4. FAVORITES
   Save your favorite games directly to your browser's persistent storage.

5. RESPONSIVE DESIGN
   A fluid, mobile-first grid layout that works on any device.

⚙️ INSTALLATION
---------------
1. Obtain an API Key from RAWG.io.
2. Open script.js and add your key to the API_KEY variable.
3. Open index.html in any modern browser.
